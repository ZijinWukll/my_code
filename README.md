# 这是我自己学习代码用的仓库
南京大学计算机本科生，正在苦读信息与计算科学，纯萌新小白体验高中毕业入坑cs的大队伍，不知道自己能学成什么样呢？
**我想用这个仓库来保存我的学习成长路上写的所有代码**，像是一个记录成长的本子一样，等四年后看看自己到底能有哪些进步。
希望不会半途而废吧。
## 大一上
学代码的课只有一周一节，名字叫信息与计算科学导论实验，虽然叫导论但是实际上作业难度很高，个人认为不太适合初学者入门，完全就是学长炫技欺负新人。
一个学期一共有6次作业，从OJ1到OJ6我都认真完成它，虽然真的很多地方完全不会写都是靠ai帮忙写代码，起码我还是一直在努力跟上节奏（vibe coding也是很重要的技能对吧）。 
- 12.16 开始创建了一些小型的项目。这是一个值得纪念的日子，起因是想给一个好朋友做一个有创意的生日祝福，最终我设计了一个网页，是project里的birthday_wish文件夹，这应该算是学了这么久计算机以来的的第一个项目吧，算然全都是前端没有后端但还是很有成就感，我把文件夹放在主仓库里了，同时专门备份了一份在单独的仓库里以便上线部署。  
>寒假期间我要认真梳理本学期的所学所获。12.11
